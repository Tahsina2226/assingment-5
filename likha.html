Important FAQ

What is the Document Object Model (DOM)?

The **DOM (Document Object Model)** is a programming interface for web documents,
allowing you to manipulate the structure, style, and content of a webpage by treating
it as a tree of objects that can be accessed and modified using JavaScript.

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, you can use a variety of methods,
depending on the element's attributes (e.g., ID, class, tag). Here are the most common methods:

getElementById: Selects an element by its ID:

const element = document.getElementById('elementId');

getElementsByClassName: Selects all elements with a specific class. It returns an HTMLCollection (which is like an array).

const elements = document.getElementsByClassName('className');

querySelector:Selects the first element that matches a CSS selector.

const element = document.querySelector('.className');

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events
on its child elements via event bubbling. It improves performance and simplifies handling dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute() or by directly setting properties like element.src.
For styles, use element.style.propertyName to modify specific CSS properties. You can also toggle classes
with element.classList.add() or remove().
